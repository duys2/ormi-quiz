/* ---------------- 답 ---------------- */
1. this는 자바에서 현재 객체를 참조하는 키워드 (인스턴스(객체) 나 자신을 가르키는 예약어)
    - 인스턴스 메서드 내에서 사용되고 해당 메서드를 호출한 객체 자신을 가리킨다.
    - this를 사용하여 현재 객체의 멤버 변수와 메서드에 접근할 수 있다.
    - 객체의 멤버 변수와 메서드에 접근, 호출하기 위해 사용한다.
2. final 필드는 객체마다 저장되고 생성자 매개변수로 여러가지 값을 가질 수 있기 때문에 상수가 될 수 없고,
static final은 객체마다 저장할 필요가 없이 공용으로 선언하여 사용하기 때문에 상수라고 할 수 있다.
3-1. 변수 선언과 동시에 초기화
3-2. 생성자를 통해 초기화
4. private, default, protected, public
- public: 모든 클래스에서 접근 가능
- protected: 동일한 패키지 혹은 자식 클래스에서 접근 가능
- default: 기본값. 동일한 패키지의 클래스에서 접근 가능
- private: 같은 클래스 내에서만 접근 가능
5. 데이터의 무결성을 보장하기 위해 private로 선언한 변수에 접근, 설정하기 위해 사용
- 객체의 데이터를 객체 외부에서 직접적으로 접근하는 것을 막음으로써 객체의 무결성이 깨지는 것을 막을 수 있다.
